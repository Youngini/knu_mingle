<h1> 💡 서비스 소개
<h2>서비스 개요</h2>
<p>
    KNU Mingle은 경북대학교에 재학 중인 외국인 학생들을 위한 맞춤형 커뮤니티 플랫폼입니다. 
    <br>이 앱은 학교 생활에 필요한 다양한 정보와 물품 거래, 일상 생활에 관한 지식을 공유할 수 있는 공간을 제공합니다.
</p>
<ul>
    <li>
        <strong>리뷰 게시판</strong>: 사용자는 기숙사, 학교 시설, 수업 등 다양한 키워드를 통해 소중한 경험을 공유하고, 다른 학생들의 리뷰를 쉽게 조회할 수 있습니다.
    </li>
    <li>
        <strong>장터 게시판</strong>: 학생 간 물품 거래 또는 나눔 활동에 참여할 수 있는 온라인 장터를 통해 필요한 물건을 쉽게 사고팔거나 나누는 기회를 제공합니다.
    </li>
    <li>
        <strong>데일리 맵</strong>: 지도에 등록된 데이터를 활용하여 일상 생활에 필수적인 정보를 얻을 수 있으며, 외국인 학생들이 쉽게 적응할 수 있도록 돕습니다.
    </li>
</ul>
<p>
    이 서비스는 외국인 학생들 간의 네트워크를 강화하고, 보다 편리하고 원활한 학교 생활을 지원하기 위해 개발되었습니다.
</p>


<h2>서비스 제안 배경</h2>
<ol>
    <li>
        <strong>외국인 학생들을 위한 커뮤니티 부재</strong>
        <ul>
            <li>외국 학교로 파견되는 한국인 학생들을 위한 커뮤니티(<a href="https://cafe.naver.com/knuoia" target="_blank">국제교류처 카페</a>)는 활성화되어 있지만, 외국인 학생들을 위한 전용 커뮤니티는 부족합니다.</li>
            <li>외국인 학생들이 학교 생활이나 한국 생활 전반에 대한 정보를 나누고, 물건을 쉽게 사고팔 수 있는 온라인 공간의 필요성이 대두되고 있습니다.</li>
        </ul>
    </li>
    <li>
        <strong>기존의 국내 앱 이용의 어려움</strong>
        <ul>
            <li>국내에서 널리 사용되는 앱인 에브리타임, 당근마켓, 중고나라 등은 외국인 학생들이 접근하기 어렵습니다.</li>
            <li>이러한 서비스는 주로 한국어로 제공되며, 다국어 지원이 부족하여 외국인 학생들이 정보를 이해하고 활용하는 데 어려움을 겪고 있습니다.</li>
        </ul>
    </li>
    <li>
        <strong>KNU 버디 프로그램의 한계</strong>
        <ul>
            <li>한국인 학생(버디)이 모든 외국인 학생들의 생활 전반을 돕는 데 어려움이 있습니다.</li>
            <li>특히, 외국인 학생들의 질문이나 요구 사항에 즉각적으로 대응하기 힘든 상황이 종종 발생합니다.</li>
        </ul>
    </li>
    <li>
        <strong>실제 교환학생 경험에 기반한 필요성</strong>
        <ul>
            <li>팀원들 중 교환학생을 경험한 학생들의 의견을 바탕으로, 외국인 학생들을 위한 정보 공유 커뮤니티가 필요하다는 아이디어가 제시되었습니다.</li>
            <li>실제로 KNU 버디 프로그램에서 같은 국적의 학생을 연결해 달라는 요청을 받은 경험이 바탕이 되어, 외국인 학생 간의 네트워크가 중요하다는 인식이 생겼습니다.</li>
        </ul>

## 타서비스와의 차별점
1. 쉽고 직관적인 UI/UX -> 외국인 학생들이 이용하는 데에 어려움 최소화
2. 서비스 대상 자체를 '외국인 학생'으로 설정 -> 기본 지원 언어가 한국어가 아닌 영어
3. 공공데이터 + 지도 API 활용 -> 커뮤니티 기능뿐만 아니라 일상 생활에 필요한 정보 제공

# 📱 구현 내용 및 결과물
서비스의 실제 구현 내용과 결과물을 기재한다.

1. 일반 로그인 -> 인증, JWT 토큰 -> 인가
2. 리뷰 게시판
3. 장터 게시판
4. Daily Map
5. 공지함

## 구현 방식
어떤 언어와 프레임워크를 사용했는지, 만약 배포를 진행했다면 환경은 어떤지 등을 기재
ex) 프론트엔드로는 TypeScript와 React를 사용, 백엔드로는 Java와 Spring을 사용. AWS를 통해 배포를 진행

## 향후 개선 혹은 발전 방안
1. 대구 내 타 대학교 파견 학생들과의 교류 기능 확장
   - 경북대학교뿐만 아니라, 대구 지역 내 타 대학교로 파견된 외국인 학생들과의 정보 공유 및 커뮤니티 교류를 가능하게 하여 서비스 범위를 확대.
2. 주기적인 공공데이터 추가를 통해 Daily Map 기능의 지속적인 업데이트
3. 국제교류처 공식 앱으로 활용
   - 경북대학교 국제교류처의 공식 앱으로서, 버디 프로그램, 외국인 학생 오리엔테이션(OT) 등 다양한 국제교류 활동을 지원하는 플랫폼으로 기능할 수 있도록 확장.
